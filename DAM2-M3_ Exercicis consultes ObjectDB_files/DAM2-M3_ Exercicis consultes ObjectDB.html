    Exercicis consultes ObjectDB

    INICIALS

    1. Consultar total, màxim, mínima i mitjana puntuacions de preguntes
    2. Consultar preguntes que continguin el text "cert" ordenades per puntuació
    3. Consultar preguntes verdader o fals que continguin el text "cert" i puntuació inferior o igual a 1
    4. Consultar preguntes que text de més de 50 caràcters, ordenades per mida del text
    5. Consultar total de preguntes agrupades per puntuació
    6. Consultar els noms dels estudiants dels lliuraments ordenats per nom
    7. Consultar els noms dels estudiants en majúscules i els cognoms en minúscules ordenats per edat
    8. Consultar 10 primers caràcters del text de les preguntes ordenats per puntuació descendent
    9. Consultar 10 darrers caràcters del text de les preguntes ordenats per puntuació ascendent
    10. Consulta el text de les preguntes traient espais del principi i del final
    11. Consulta el text de les preguntes traient els caràcters ': ' del final
    12. Consulta el text de les preguntes de Verdader Fals traient el caràcter '-' del principi
    13. Consultar els lliuraments de tots els alumnes amb nom el següent patró '_a%'
    14. Consultar les preguntes amb puntuació entre 1 i 1.5 inclosos ordenades per text
    15. Consultar exàmens (mòdul i títol de la unitat formativa) i la suma de les puntuacions de les seves preguntes ordenats per puntuació descendent

    PARÀMETRES

    1. Consultar total de preguntes agrupades per puntuació sempre que el total de preguntes sigui > PARAM, on PARAM és un paràmetre variable
    2. Consultar preguntes verdader o fals que continguin el text PARAM1 i puntuació inferior o igual a PARAM2, on PARAM1 i PARAM2 és un paràmetre variable (Feu servir la opció de paràmetres numerats: 1, 2, ..)
    3. Consultar preguntes que text de més de PARAM caràcters, ordenades per mida del text, on PARAM és un paràmetre variable
    4. Consultar les preguntes amb puntuació entre 1 i 1.5 inclosos ordenades per text



    PAGINACIÓ

    1. Per una mida de pàgina de 3, obtenir la primera pàgina de la consulta dels noms, cognoms i edat dels estudiants ordenats per longitud de nom+cognoms
    2. Obtenir la segona pagina de la consulta anterior
    3. Obtenir la darrera pagina de la consulta anterior


    JOIN

    1. Consultar el nom, cognoms i edat dels alumnes i la nota dels diferents Lliuraments ordenats per nota
    2. Consultar TOTS els exàmens i les seves preguntes si escau. Cal mostrar cicle i número de la Unitat Formativa de l'examen i el text i la puntuació de la pregunta. Ordenat per cicle i número
    3. Consultar preguntes dels exàmens que tinguin puntuació superior a PARAM1, on PARAM és un paràmetre variable. Cal mostrar cicle i número de la Unitat Formativa de l'examen i el text i la puntuació de la pregunta. Ordenat per puntuació descendent
    4. Consultar lliuraments d'exàmens amb més de PARAM preguntes, on PARAM és un paràmetre variable. Cal mostrar cicle i número de la Unitat Formativa de l'examen, el total de preguntes de l'examen i el nom,cognoms i edat de l'alumne que ha fet el lliurament. Ordenat pel total de preguntes de l'examen i per nom i cognoms de l'alumne.



    NAMED QUERYS

    1. Crear NamedQuerys "findAll" a totes les classes per consultar totes les instàncies: Estudiant, Lliurament, Examen, UnitatFormativa, Pregunta, PreguntaOberta, PreguntaVerdaderFals
    2. Crear NamedQuery "findByEdat" a la classe Estudiant que permeti consultar estudiants majors d'un cert valor indicat per paràmetre
    3. Crear NamedQuery "findBetweenNotes" a la classe Lliurament que permeti consultar lliuraments amb nota entre 2 valors indicats per paràmetre

    SOLUCIONS

    1. CONSULTA 1 - total, màxim, mínima i mitjana puntuacions

strQuery = "SELECT COUNT(p), MAX(p.puntuacio), "
 	 + "MIN(p.puntuacio),  AVG(p.puntuacio) FROM Pregunta p";
TypedQuery<Object[]> query = em.createQuery(strQuery, Object[].class);
List<Object[]> resultats = query.getResultList();
for (Object[] resultat : resultats) {
	System.out.println(resultat[0]);
	System.out.println(resultat[1]);
	System.out.println(resultat[2]);
	System.out.println(resultat[3]);
}



    2. CONSULTA 2 - Consultar preguntes que continguin el text \"cert\" ordenades per puntuació


strQuery = "SELECT p FROM Pregunta p "
	 + "WHERE p.text LIKE '%cert%' ORDER BY p.puntuacio ";
TypedQuery query = em.createQuery(strQuery, Pregunta.class);
List resultats = query.getResultList();
for (Pregunta resultat : resultats) {
	System.out.println(resultat.text);
}

    3. CONSULTA 3 - Consultar preguntes verdader o fals que continguin el text \"cert\" i puntuació inferior o igual a 1

strQuery = "SELECT p FROM PreguntaVerdaderFals p "
	 + "WHERE p.text LIKE '%cert%' AND "
	 + "p.puntuacio <= 1";
TypedQuery query = em.createQuery(strQuery, Pregunta.class);
List resultats = query.getResultList();

for (Pregunta resultat : resultats) {
	System.out.println(resultat.getText());
}


    1. CONSULTA PARAMS 1 - Consultar total de preguntes agrupades per puntuació
		       sempre que el total de preguntes sigui > PARAM,
		       on PARAM és un paràmetre variable

strQuery  = "SELECT p.puntuacio, COUNT(p) FROM Pregunta p ";
strQuery += " GROUP BY p.puntuacio";
strQuery += " HAVING(COUNT(p)) > :total ";
TypedQuery<Object[]> query = em.createQuery(strQuery, Object[].class);
query.setParameter("total", 0);
List<Object[]> resultats = query.getResultList();

for (Object[] resultat : resultats) {
	System.out.println(resultat[0]+" "+resultat[1]);
}
}



    2. CONSULTA PARAMS 2 - Consultar total de preguntes agrupades per puntuació
		       sempre que el total de preguntes sigui > PARAM,
		       on PARAM és un paràmetre variable

strQuery  = "SELECT p, COUNT(p.puntuacio) FROM Pregunta p ";
strQuery += " GROUP BY p, COUNT(p.puntuacio)";
strQuery += " HAVING(COUNT(p.puntuacio)) > :param ";
TypedQuery<Object[]> query = em.createQuery(strQuery, Object[].class);
    query.setParameter("param", 2);
List<Object[]> resultats = query.getResultList();

for (Object[] resultat : resultats) {
	System.out.println(resultat[0]+" "+resultat[1]);
}




    1. CONSULTA PAGINACIO 1 - Per una mida de pàgina de 3, obtenir la primera pàgina
		          de la consulta dels noms, cognoms i edat dels estudiants
		          ordenats per longitud de nom+cognoms

strQuery  = "SELECT e.nom, e.cognoms, e.edat FROM Estudiant e ";
strQuery += " ORDER BY LENGTH(CONCAT(nom, cognoms)) ";

    1. CONSULTA JOIN 1 - Consultar el nom, cognoms i edat dels alumnes
		     i la nota dels diferents Lliuraments
		     ordenats per nota

strQuery  = "SELECT e.nom, e.cognoms, e.edat, l.nota  ";
strQuery += " FROM Lliurament l INNER JOIN l.estudiant e ";
strQuery += " ORDER BY l.nota ";
TypedQuery<Object[]> query = em.createQuery(strQuery, Object[].class);
List<Object[]> resultats = query.getResultList();

	System.out.println(resultat[0]+" "+resultat[1]+" "+resultat[2]+" "+resultat[3]);
}
